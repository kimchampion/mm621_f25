World Happiness Report

This visualization shows the happiness score by region from the World Happiness Report for 2015-2019, using emojis to represent each region’s overall mood. Hovering over a region reveals its countries and happiness rankings, connecting regional averages to country-level details. It highlights global patterns and differences in happiness across the world. For more detailed information about this data set and report, go to the World Happiness Reporta> on kraggle.com
